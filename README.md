# Inventory System (Java)

This is a simple inventory management system developed in Java.

## Features
- Add products
- Show all products
- Sell products (update stock)
- Save data to a .txt file

## Concepts Used
- Object-Oriented Programming (OOP)
- ArrayList
- File handling (FileWriter)
- User input (Scanner)

## How to run
1. Compile the files
2. Run the Main class
3. Use the menu in console
